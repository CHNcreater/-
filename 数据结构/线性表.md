# 线性表
线性表分类：顺序存储和链式存储，其中链式存储还可以分为单链表、双链表、循环链表、静态链表等不同的表示方式。
每种表示方式有其各自的数据结构和基本操作。

## 顺序存储
  ```
  typedef struct{
    int data[MAXSIZE];
    int length;
  }SqList;//顺序线性存储表
  typedef struct{
    int *data;
    int MAXSIZE,length;//数组最大容量和当前数量
  }SeqList;//动态申请线性表
  ```
  特点：
  1. 随机存取，时间复杂度均为O(1)；
  2. 删除插入操作复杂，最好时间复杂度是O(1)，最坏时间复杂度是O(n)，平均时间复杂度是O(n)
  3. 按值查找的时间复杂度最好情况是O(1)，最坏情况是O(n)，平均时间复杂度是O(n)
